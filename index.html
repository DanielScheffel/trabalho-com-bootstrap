Sobre o Curso

Como funciona o curso?

O curso vai do básico ao avançado. Sendo assim, você pode escolher entre duas trilhas: uma que ensina a base necessária sobre programação e outra aonde você aprende a programar com estruturas complexas.

Sempre que uma nova tecnologia for ensinada, você será levado a criar projetos simples para que consiga absorver tudo o que é necessário. Só depois irá para um projeto complexo.

Domine o Frontend com ReactJS: aprenda a fazer componentes reutilizáveis para economizar tempo e esforço. Trabalhe com uma estrutura robusta que permite criar interfaces interativas e responsivas.

Domine o Backend com Node.JS: aprenda a criar uma API de alta performance que consegue lidar com um grande número de solicitações. Trabalhe num projeto prático que conta com uma integração ao banco de dados e um sistema de autenticação.

Módulos

Conheça os Módulos do Curso

01 Instalando Dependências

Neste módulo você irá instalar as ferramentas necessárias para programar, como o Visual Studio Code, que acelera o processo de desenvolvimento.

02 HTML (Guia Completo)

Conheça nesse módulo o HTML, que é a estrutura básica de qualquer site. Aprenda sobre todas as Tags necessárias para a construção de páginas.

03 CSS (Introdução)

O CSS é responsável pela estilização de páginas. Aprenda todo o princípio básico para começar a estilizar e também como fazer Layouts responsivos (para qualquer tamanho de tela).

04 CSS (Flexbox)

O Flexbox, que é uma técnica de estruturação de Layouts, facilita muito na hora de estilizar. Aprenda nesse módulo a mexer no CSS com mais eficiência.

05 Página Completa (Login Page)

Depois de ver sobre HTML e CSS, é necessário praticar. Nesse módulo você vai aprender a criar do zero uma página de Login completa.

06 JavaScript (Guia Completo)

O JavaScript permite implementar comportamentos complexos em páginas. Aprenda sobre variáveis, tipos de dados, laços de repetição e muito mais.

07 Projeto com HTML, CSS e JS

Depois de aprender sobre as três principais tecnologias usadas na Web, é hora de fazer um projeto simples para fixar e entender melhor tudo o que foi aprendido até aqui.

08 Iniciando com React

Esse será o seu primeiro contato com o React. O React é uma biblioteca JavaScript que permite criar página dinâmicas de maneira eficiente. Ele é um facilitador para quem programar para a Web.

09 Conhecendo o Redux

O Redux é uma biblioteca JavaScript que se integra com React. Ele permite criar estruturas complexas e organizadas, além de gerenciar o estado de um sistema Web.

10 Projeto com React + Redux

É hora de praticar com um projeto mais abrangente. Depois de entender bem sobre React e Redux, aprenda aqui a fazer um projeto com maior complexidade.

11 React Hooks

Os Hooks (Ganchos) são recursos que o React fornece para gerenciar variáveis e ciclos de vida de componentes. Com ele é possível criar componentes com menos código.

13 To Do List - Backend (Node.JS + MongoDB)

Nesse módulo você terá o seu primeiro contato com Node.JS e MongoDB. Aprenda aqui a criar uma API completa que se integra com banco de dados.

14 To Do List - Frontend (React + Redux)

É hora de pôr a mão na massa e trabalhar com um Sistema Web completo. Aprenda a trabalhar com uma estrutura robusta, com dados que são alimentados por uma API.

15 To Do List > Disponibilizando Online

Aprenda a disponibilizar Online, gratuitamente, o sistema que foi criado nos módulos anteriores.

16 Sites Recomentados

Nesse módulo você vai conhecer as documentações oficiais das tecnologias usadas no curso. Saiba como acessar cada documentação e como tirar o melhor proveito delas.

Sobre o Projeto Principal

Os módulos 13, 14 e 15 apresentam um projeto completo com Backend e Frontend.

A maioria das empresas de sistemas utilizam tanto Backend quanto Frontend. Essa arquitetura é robusta, segura e permite criar uma interface de usuário interativa e responsiva. Além disso, essa estrutura possibilita que as equipes de desenvolvimento trabalhem com mais eficiência e colaboração.
